```<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <!--input엔 다양한 type속성들이 있습니다-->
    <input type="text"> 한줄짜리 텍스트를 입력할 수 있는 텍스트 박스를 넣는다
    <input type="password"> 비밀번호를 입력할수 있는 필드를 넣습니다
    <input type="search"> 검색할 때 입력하는 필드를 넣습니다
    <input type="url"> url주소를 입력할수 있는 필드를 넣습니다
    <input type="email"> 이메일 주소를 입력할 수 있는 필드를 넣습니다
    <input type="tel"> 전화번호를 입력할 수 있는 필드를 넣습니다
    <input type="checkbox"> 주어진 항목에서 2개이상 선택할 수 있는 체크박스를 넣습니다
    <input type="radio"> 주어진 여러항목에서 1개만 선택할수 있는 라디오버튼을 넣습니다
    <input type="number"> 숫자를 조절할 수 있는 스핀 박스를 넣습니다
    <input type="range"> 숫자를 조절할 수 있는 슬라이드 막대를 넣습니다
    <input type="date"> 사용자 지역을 기준으로 날짜(연,월,일)를 넣습니다
    <input type="month"> 사용자 지역을 기준으로 날짜(연,월)를 넣습니다
    <input type="week">사용자 지역을 기준으로 날짜(연,주)를 넣습니다
    <input type="time"> 사용자 지역을 기준으로 날짜(시간,분,초,분할 초)를 넣습니다
    <input type="datetime"> 국제 표준시(UTC)로 설정된 날짜와 시간(연,월,일,시,분,초,분할 초)를 넣습니다
    <input type="datetime-local"> 사용자 지역을 기준으로 날짜와 시간(연,월,일,시,분,초,분할 초)를 넣습니다
    <input type="submit"> 전송 버튼을 넣습니다
    <input type="reset"> 리셋 버튼을 넣습니다
    <input type="image"> submit버튼 대신 사용할 이미지를 넣습니다
    <input type="button"> 일반 버튼을 넣습니다
    <input type="file"> 파일을 첨부할 수 잇는 버튼을 넣습니다
    <input type="hidden"> 사용자에게는 보이지 않지만 서버로 넘겨주는 값이 잇는 필드를 만듭니다
</body>
</html>```
